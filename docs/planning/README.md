# Planning Archive

These documents capture the project's early strategic and architectural
thinking. They're kept for historical context, but several describe
features or infrastructure that were never built, or that have since
changed (e.g. pgvector/embeddings were stubbed out, the frontend migrated
from Vite to Next.js). Each file has a banner noting what's stale.

For the current, accurate picture of the project, see the root
[README.md](../../README.md) and the operational docs at the repo root
(`CHANGELOG.md`, `DATABASE_MIGRATION_GUIDE.md`, `CONTENT_RATINGS.md`,
`MULTI_DIMENSIONAL_RATINGS.md`).
